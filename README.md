# Claude Session Manager

[Claude Code](https://claude.com/claude-code) の会話履歴・メモリ・エージェントを管理する VS Code 拡張機能。

ブックマーク、タグ付け、メモ、Markdownプレビュー、子エージェントのツリー表示、組織図など、Claude Code本体にない会話・エージェント管理機能を提供します。

---

## 主な機能

### 会話管理

- **会話一覧** — 全プロジェクトの会話を日付別（今日/昨日/今週/今月/それ以前）に自動分類
- **モデル表示** — モデル頭文字（Ｏ=Opus / Ｓ=Sonnet / Ｈ=Haiku）とメッセージ件数を表示
- **プレビュー** — チャット形式で会話内容を表示。Markdownレンダリング対応（コードブロック・テーブル・見出し・リスト）
- **リンクのクリック対応** — プレビュー内のURLはブラウザ、Windowsファイルパスはエディタで開く（行番号ジャンプ付き）
- **ブックマーク** — 大事な会話をお気に入り登録。専用セクションに一覧表示
- **タグ** — 自由なカテゴリで分類。タグ別グループ表示。プレビュー画面から直接追加/削除
- **メモ** — 各会話に役割や目的のメモを記入可能。500ms遅延で自動保存
- **リネーム** — 会話にわかりやすい名前を設定。Claude Code側と双方向同期
- **検索** — タイトル・プロジェクト・ブランチ名でフィルタ
- **会話内検索** — プレビュー内のメッセージをキーワードで絞り込み
- **Claude Codeで開く** — 右クリックからVS CodeのClaude Codeパネルで会話を再開（Antigravity等フォーク対応）
- **セッションIDコピー** — 右クリックからセッションIDをワンクリックでクリップボードにコピー
- **他プロジェクト色分け** — 現在のワークスペース以外のセッションは薄い色で表示

### ライブセッション検出

- Claude Codeで現在使用中のセッションをリアルタイム検出
- 使用中の会話は緑の **●** アイコンで表示
- `~/.claude/sessions/` を監視して自動更新

### 子エージェントのツリー表示

- セッションに紐づいた子エージェント（`subagents/` フォルダ内の JSONL）を自動検出
- 親セッションを展開すると、配下の子エージェントをツリー表示
- `meta.json` からエージェントタイプ・説明を読み込み、タイプ別アイコンで区別

| タイプ | アイコン | ラベル |
|--------|---------|--------|
| Explore | 🔍 search（青） | 🔍探索 |
| Plan | 📋 notebook（紫） | 📋計画 |
| general-purpose | 🔧 tools（橙） | 🔧汎用 |
| claude-code-guide | 📖 book（緑） | 📖ガイド |

### エージェント管理

- **エージェント登録** — セッションを右クリック → 「エージェントとして登録」。名前・役割・モデル・ルールファイル・親エージェント・許可ツールを設定。`~/.claude/session-manager.json` に保存
- **エージェント役割タグ** — `[agent]部署名` 形式のタグで、セッションにエージェント役割を付与・削除
- **ルールファイルを開く** — 右クリックからエージェントのルールファイル（.md）を直接編集
- **プレビューヘッダのバッジ** — エージェント登録済みのセッションはプレビュー上部にモデル・名前・役割を表示

### エージェント組織図

- ステータスバー（左下）に現在のエージェント稼働数を表示（例: `⬟ 2/5 アクティブ`）
- クリックすると Webview パネルで組織図を表示
- 各エージェントカードにモデルバッジ・役割・セッション状態（ライブ/オフライン）を表示
- **▶ ボタン** でカードから直接セッションをプレビュー

### メモリ管理

- **メモリ一覧** — タイプ別バッジ（user / feedback / project / reference）付きで表示
- **容量表示** — プログレスバー付きインデックス使用率（最大200行）
- **プレビュー・編集・統合・抽出・削除**
- **OneDrive対応** — 日本語を含むOneDriveパスのプロジェクト名を正しく表示

## 動作要件

- VS Code 1.85 以上（Antigravity 等の VS Code フォークでも動作）
- [Claude Code for VS Code](https://marketplace.visualstudio.com/items?itemName=anthropic.claude-code) がインストール済み

## 使い方

1. VS Code 左のアクティビティバーに 💬 アイコンが表示される
2. クリックするとサイドバーに4セクション（会話一覧・ブックマーク・タグ・メモリ管理）が表示
3. 会話をクリックでプレビュー表示
4. 会話を右クリックでコンテキストメニュー（ブックマーク・タグ・リネーム・IDコピー・エージェント登録など）
5. ステータスバー（左下）のエージェント表示をクリックで組織図を開く

詳しい使い方は同梱の `guide.html` をブラウザで開いて確認できます（コマンドパレット → `Claude: 使い方ガイドを開く`）。

## コマンド一覧

| コマンド | 説明 |
|---|---|
| `Claude: 会話一覧を更新` | セッション一覧を再読み込み |
| `Claude: 会話を検索` | キーワードフィルタを設定 |
| `Claude: セッションIDをコピー` | IDをクリップボードにコピー |
| `Claude: Claude Codeで開く` | 選択した会話をClaude Codeで再開 |
| `Claude: ブックマークに追加/解除` | ブックマーク操作 |
| `Claude: タグを追加` | タグを新規または既存から選択して付与 |
| `Claude: リネーム` | 会話名を変更 |
| `Claude: エージェントとして登録` | セッションにエージェント設定を紐づけ |
| `Claude: ルールファイルを編集` | エージェントのルールファイルを開く |
| `Claude: エージェント役割を設定` | `[agent]` タグを付与・削除 |
| `Claude: エージェント組織図を開く` | 組織図 Webview を表示 |
| `Claude: 使い方ガイドを開く` | guide.html をブラウザで開く |
