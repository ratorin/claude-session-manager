# 更新履歴

## v0.1.8 (2026-04-01)
### エージェント管理
- **エージェント登録** — セッションを右クリック →「エージェントとして登録」で名前・役割・モデル・ルールファイル・親エージェント・許可ツールを設定。`~/.claude/session-manager.json` に永続保存
- **エージェント役割タグ** — `[agent]部署名` 形式のタグでセッションにエージェント役割を付与・削除
- **ルールファイルを編集** — 右クリックから登録済みエージェントのルールファイル（.md）を直接編集
- **ステータスバー** — 左下にエージェント稼働数をリアルタイム表示（例: `⬟ 2/5 アクティブ`）。クリックで組織図を開く
- **エージェント組織図 Webview** — 階層構造・モデルバッジ・セッション状態（ライブ/オフライン）を一覧表示。▶ボタンからセッションをプレビュー

### 子エージェントのツリー表示
- `subagents/` フォルダ内の JSONL を自動検出・読み込み
- `isSidechain: true` のセッションを子エージェントとして識別し、親セッション配下にツリー表示
- `meta.json` から `agentType` / `description` を読み込み、タイプ別アイコンで表示
  - 🔍探索（Explore）/ 📋計画（Plan）/ 🔧汎用（general-purpose）/ 📖ガイド（claude-code-guide）

### プレビュー強化
- **Markdownレンダリング** — コードブロック・テーブル・見出し・太字・リスト・インラインコードに対応
- **リンクのクリック対応**
  - URLリンク（`https://...` または `[text](url)` 構文）→ ブラウザで開く
  - Windowsファイルパス（`C:\path\to\file.ts:42` 形式）→ エディタで開く（行番号ジャンプ付き）
  - プレースホルダー方式で二重マッチを防止
- **エージェントバッジ** — 登録済みエージェントのセッションはプレビュー上部にモデル・名前・役割を表示

### その他
- **セッションIDコピー** — 右クリックからIDをワンクリックでクリップボードにコピー
- **VS Code フォーク対応** — Antigravity 等の VS Code フォークで「Claude Codeで開く」が正しく動作
- **README 全面改訂** — 全機能の説明・コマンド一覧・アイコン凡例を整備

## v0.1.0 (2026-03-28)
初回リリース

### 会話管理
- 全プロジェクトの会話を日付別（今日/昨日/今週/今月/それ以前）に自動分類
- チャット形式プレビュー（ヘッダ情報+メモ欄+会話内容）
- ブックマーク・タグ・メモ・リネーム
- タイトル・プロジェクト・ブランチ名で検索
- 会話内キーワード検索
- 右クリックからClaude Codeで会話を再開
- モデル頭文字（Ｏ/Ｓ/Ｈ）とメッセージ件数を表示
- リネーム時は元メッセージをdescriptionに表示
- 他プロジェクトのセッションを薄い色で表示
- Claude Code自動生成タイトル（ai-title）に対応

### ライブセッション検出
- Claude Codeで使用中のセッションをリアルタイム検出（緑●アイコン）

### メモリ管理
- タイプ別バッジ付きファイル一覧
- プログレスバー付きインデックス使用率表示
- プレビュー・編集・統合・抽出・削除
- 各ファイルの行数表示

### アイコン体系
- 通常: primitive-dot（小さい点）
- ブックマーク: star-full（黄）
- 利用中: circle-filled（緑）
- プレビュー中: eye
- プレビュー+利用中: target（緑）
- モデルはアイコンではなくラベル頭文字（Ｏ/Ｓ/Ｈ）で表示

### その他
- Claudeオレンジの拡張機能アイコン
- 使い方ガイド（guide.html）をブラウザで開くコマンド
- Claude風配色のガイドページ
