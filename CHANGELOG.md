# 更新履歴

## v0.1.7 (2026-03-31)
- プレビュー内のリンクをクリック可能に
  - URLリンク → ブラウザで開く
  - Windowsファイルパス → エディタで開く（行番号付き対応）
  - Markdownリンク `[text](url)` 構文に対応
- プレースホルダー方式でリンクの二重マッチを防止

## v0.1.6 (2026-03-31)
- **Phase 2: 子エージェント表示対応**
  - `subagents/` フォルダのJSONLを読み込み対象に追加
  - `isSidechain: true` のセッションを子エージェントとして識別
  - 親セッションを展開すると配下の子エージェントをツリー表示
  - 子エージェントにタイプ別アイコン・ラベル（🔍探索 / 📋計画 / 🔧汎用 / 📖ガイド）
  - `meta.json` から `agentType` と `description` を読み込み
- エージェント設定を `session-manager.json` に保存（`registerAgent` / `editRuleFile` コマンド追加）
- セッションプレビューでMarkdownレンダリング対応
- ステータスバーにエージェント稼働状況を表示
- 組織図から `▶` ボタンでセッションをプレビュー表示

## v0.1.5 (2026-03-31)

## v0.1.0 (2026-03-28)
初回リリース

### 会話管理
- 全プロジェクトの会話を日付別（今日/昨日/今週/今月/それ以前）に自動分類
- チャット形式プレビュー（ヘッダ情報+メモ欄+会話内容）
- ブックマーク・タグ・メモ・リネーム
- タイトル・プロジェクト・ブランチ名で検索
- 会話内キーワード検索
- 右クリックからClaude Codeで会話を再開
- モデル頭文字（Ｏ/Ｓ/Ｈ）とメッセージ件数を表示
- リネーム時は元メッセージをdescriptionに表示
- 他プロジェクトのセッションを薄い色で表示
- Claude Code自動生成タイトル（ai-title）に対応

### ライブセッション検出
- Claude Codeで使用中のセッションをリアルタイム検出（緑●アイコン）

### メモリ管理
- タイプ別バッジ付きファイル一覧
- プログレスバー付きインデックス使用率表示
- プレビュー・編集・統合・抽出・削除
- 各ファイルの行数表示

### アイコン体系
- 通常: primitive-dot（小さい点）
- ブックマーク: star-full（黄）
- 利用中: circle-filled（緑）
- プレビュー中: eye
- プレビュー+利用中: target（緑）
- モデルはアイコンではなくラベル頭文字（Ｏ/Ｓ/Ｈ）で表示

### その他
- Claudeオレンジの拡張機能アイコン
- 使い方ガイド（guide.html）をブラウザで開くコマンド
- Claude風配色のガイドページ
